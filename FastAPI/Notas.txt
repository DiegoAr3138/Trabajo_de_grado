1. Creamos un entorno virtual
    Usamos el siguiente comando para crear el entorno de trabajo
    y no tener que instalar todos los modulos en global

        python3.11 - m venv venv

    Con el comando anterior creamos el entorno virutal 

2. Activacion del entorno virtual usamos el siguiente comando

        source venv/Scripts/activate 

    Para desactivarlo usamos el comando 
    
    deactivate


3. Correr el FastAPI
    para poder correr el FastAPI usamos el comando uvicorn en la terminal:
    --port : esto me ayuda a seleccionar el puerto
    --reload : esto me ayuda a que la pagina se recargue sola cada vez que guardemos cambios
    --host : esto ayuda para que podamos usarla desde otros dispositivos 

        uvicorn (nombre del archivo) : (nombre de la app) --host (host) --port (puerto) --reload

        ejemplo:

        Revisar el comando host
        
            uvicorn src.main:app --host 0.0.0.0 --port 5000 --reload

            uvicorn src.main:app --port 5000 --reload

    Una vez ejecutado esto nos dara un puerto al cual podemos acceder para desactivarlo lo ahcemos
    con ctrl + C
